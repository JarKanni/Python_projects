# Wind Check


# Explanation
Asks user for zip code, then checks NOAA's weekly forecasts for wind greater than 10mph. If so, lets user know when to expect it to roll in.
 
 
# Requirements
Packages used:
- noaa_sdk
- pandas
- statistics
